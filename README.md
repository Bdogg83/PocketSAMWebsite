# PocketSAM Website

Next.js marketing website for PocketSAM — separate from the Flutter app in `PocketSAM/`.

## Run locally

```powershell
cd C:\Users\bramey\VSCODE\PocketSAMWebsite
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```powershell
npm run build
npm start
```

## Assets

Place images in `public/`:

- `homepagebackground.png` — hero background
- `PocketSAM.png` — logo
